# diwali-sales-analysis
"Exploratory Data Analysis (EDA) on Diwali retail sales data using Python, Pandas, Seaborn, and Matplotlib to uncover purchasing trends across demographics, states, and product categories."

This repository contains a comprehensive Exploratory Data Analysis (EDA) on retail customer purchasing behavior during the Diwali festive season. Using a dataset of 11,239 processed transactions, the project cleans data irregularities (dropping null values and unrelated blank fields) and employs structural visualizations with Matplotlib and Seaborn.  

The analysis reveals critical consumer insights across various parameters:
Demographics: Identifying customer behaviors mapped by age groups (such as the dominant 26-35 year segment) and gender splits.  

Geographics: Highlighting top revenue-generating zones and states like Uttar Pradesh, Maharashtra, and Karnataka.  
Socio-Economic Factors: Mapping purchasing power and sales volume relative to buyer occupations and marital status.  

Product Performance: Evaluating which product categories, such as Food, Clothing, and Electronics, drive maximum transactional volume and financial value.  

These data-driven insights are designed to help consumer retail businesses optimize their inventory management, seasonal supply chains, and targeted marketing campaigns to maximize profitability during peak holiday periods.
